# Project 2 — Fix a Lecture Notebook Using OpenClaw

## Objective

In this project, students will use an **AI agent** to improve and repair a Jupyter Notebook lecture.

The AI agent used in this project is **OpenClaw**.

OpenClaw is an autonomous AI assistant that can interact with large language models to:

* edit files
* run commands
* execute coding tasks
* automate development workflows

You will use OpenClaw to **fix and improve a lecture notebook used in this course**.

This project is worth **20 points**.

# Security Tips

Do **NOT install OpenClaw on your personal computer**.

Instead run it in a **sandboxed environment**, such as:

* a **cloud server**
* a **local virtual machine**
* a **Docker container**

This avoids potential **security risks**, since AI agents may execute system commands or access files.
